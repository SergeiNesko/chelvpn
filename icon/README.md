# Иконка ChelVPN

Положи сюда файл `ic_launcher.png` — квадратное изображение минимум **512×512 px** в формате PNG.

Скрипт автоматически создаст все нужные размеры:
- 48×48 (mdpi)
- 72×72 (hdpi)
- 96×96 (xhdpi)
- 144×144 (xxhdpi)
- 192×192 (xxxhdpi)

Если файл не положить — останется стандартная иконка v2rayNG.
